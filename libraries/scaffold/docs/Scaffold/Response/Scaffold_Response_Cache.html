Scaffold_Response_Cache

Makes checking the requesting user agents cache easier. It extracts the etag and last-modified headers and allows you to compare another etag and modified time against these to determine if the cache is still valid.

It automatically fixes the HTTP_IF_MODIFIED_SINCE header issue in IE6.
